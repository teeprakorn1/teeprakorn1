## 🎓 Capstone Project (Senior Project)

### **Student Activity Tracking and Evaluation System with AI-Powered Deepfake Detection**

**Academic Year 2568 (2024-2025) | Faculty of Business Administration and Information Technology, RMUTTO**

#### 🌟 Project Overview

A comprehensive activity management platform developed to modernize student extracurricular tracking at RMUTTO, integrating cutting-edge AI technology to combat fraudulent activity submissions through automated deepfake detection.

#### 🎯 Key Achievements

- **System Complexity:** 150,000+ lines of code across 4 major components
- **AI Innovation:** 83.1% overall accuracy in deepfake detection using 10 ensemble CNN models
- **Mobile App Excellence:** 99-100% detection rate for mobile-edited forgeries
- **Production Scale:** Successfully deployed on Windows Server with Cloudflare CDN
- **Database Design:** 25 normalized tables with comprehensive audit trail
- **Security First:** JWT authentication, AES-256 encryption, HTTPS-only architecture

#### 🏗️ System Architecture

**Technology Stack:**
- **Mobile:** Flutter 3.0+ (cross-platform iOS/Android)
- **Frontend:** React.js 18 with responsive Material Design
- **Backend API:** Node.js 18 + Express.js with JWT authentication
- **AI Service:** FastAPI + TensorFlow 2.14 (DenseNet121 ensemble)
- **Database:** MySQL 8.0 + Redis Cloud caching
- **Infrastructure:** Windows Server 2022, NGINX reverse proxy, Let's Encrypt SSL
- **DevOps:** PM2 process manager, Cloudflare DDoS protection

**Key Features:**
- 📱 Cross-platform mobile app with GPS-verified check-in/out
- 🤖 AI-powered photo verification (10 ensemble CNN models)
- 💻 Comprehensive admin dashboard with real-time analytics
- 🔔 Multi-channel notifications (Push, Email, In-app)
- 📜 Digital certificate generation and management
- 🔒 Enterprise-grade security (RBAC, rate limiting, audit logs)
- 📊 Advanced reporting and data visualization
